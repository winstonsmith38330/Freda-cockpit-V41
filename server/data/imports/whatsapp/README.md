# WhatsApp import folder

Put WhatsApp export `.zip` or `.txt` files here, then commit/push to GitHub.

Recommended naming:
- `WhatsApp Chat with LA Donuts BH (reporting).zip`
- `WhatsApp Chat with LA DONUTS PN (reporting).zip`
- `WhatsApp Chat with LA DONUT TP (reporting).zip`

The app reads this folder via the WhatsApp tab button: **Refresh WhatsApp from GitHub imports**.
The refresh replaces the previous WhatsApp summaries/actions so Nicolas can control the full refresh from GitHub.
